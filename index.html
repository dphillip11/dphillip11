Projects

Newton's Cradle▼

After discovering the wonderful world of shaders and graphics programming, I couldn't help but create a raytraced version of Newton's Cradle. All the geometry is modelled using ray intersections and the reflections are exact, this is suitable for such a reflective scene. The scene is rendered using a raytracing algorithm, which is implemented in GLSL. The scene is rendered in real-time, allowing for an interactive experience. The user can control the camera using the arrow keys and zoom in and out using the scroll wheel.

The obvious benefit of using shaders and in the example WebGL is that it can be loaded directly from an html file and run in the browser, furthermore the size of the image is dictated by client and not file size, in fact, the code necessary to create it is only a few hundred lines of text. I will be learning WebGPU as this seems to be the future of web based graphics APIs.

Mandelbrot Visualisation▼

Although I have used webGl to give a demo in the browser, my work on Mandelbrot visualisation mostly took place on the CPU using Python and C. I implemented a variety of methods for rendering the M-set with varying degrees of efficiency. WebGl is limited in terms of floating point accuracy which certainly is not the case for many of the other implementations.

C was of course an excellent language for outputting still images and I actually used convolutional edge detection to automate this so that I could generate many images at increasing levels of zoom. Python initially seemed a little lethargic but withe the help of the Numba library and the use of the JIT compiler I was able to fare much better.
